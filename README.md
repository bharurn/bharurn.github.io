# Portfolio

![MiMiC](https://github.com/[username]/[reponame]/[branch]/mimic_logo.jpg

I am involved in the MiMiC project, which involves the developement of a new QM/MM software linking the Gromacs and CPMD packges by an efficient MPI/OpenMP communication scheme. I was involved in testing of the crucial communication library of the package, as well as testing the software's scaling with the largest protein system simulated so far. I was also the main developer for the MiMiCPy python wrapper around MiMiC. This allowed for easy preparation of the input files, using molecular visualization packages, and an easy to use interface to run MiMiC simulations
